# ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ ОТЧЕТА

## Как использовать файл `diploma_report_blocks.md`

1. **Откройте файл** `diploma_report_blocks.md` в любом текстовом редакторе

2. **Скопируйте блоки** по одному в ваш Word документ:
   - Каждый блок содержит 4 части:
     - [ФОТО КОДА] - здесь нужно вставить скриншот кода из вашего редактора
     - ОБЪЯСНЕНИЕ КОДА - готовый текст для копирования
     - [ФОТО РЕЗУЛЬТАТА] - здесь нужно вставить скриншот интерфейса
     - ОБЪЯСНЕНИЕ РЕЗУЛЬТАТА - готовый текст для копирования

3. **Делайте скриншоты**:
   - Код: Откройте файл в VS Code/Cursor, выделите нужный код, сделайте скриншот
   - Результат: Откройте сайт в браузере (http://localhost:3000), сделайте скриншот нужной страницы/компонента

4. **Вставляйте в Word**:
   - Скопируйте текст объяснения
   - Вставьте скриншот кода
   - Скопируйте текст объяснения результата
   - Вставьте скриншот интерфейса

5. **Форматирование в Word**:
   - Используйте стили заголовков для названий блоков
   - Добавьте подписи к скриншотам
   - Настройте отступы и интервалы для читаемости

## Структура отчета

Каждый блок следует структуре:
1. Название блока (например, "БЛОК 1: Главная страница")
2. [ФОТО КОДА] - скриншот кода
3. ОБЪЯСНЕНИЕ КОДА - детальное описание
4. [ФОТО РЕЗУЛЬТАТА] - скриншот интерфейса
5. ОБЪЯСНЕНИЕ РЕЗУЛЬТАТА - описание того, что получилось

## Рекомендации

- Делайте скриншоты в хорошем качестве
- Выделяйте важные части кода на скриншотах (можно использовать стрелки или рамки)
- Делайте скриншоты интерфейса на разных разрешениях (десктоп, планшет, мобильный)
- Добавляйте подписи к скриншотам в Word
