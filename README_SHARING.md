# Как поделиться проектом с другом (через Docker)

Этот проект настроен для быстрого запуска через Docker. Вашему другу не нужно устанавливать Python, Node.js или настраивать базу данных — всё запустится одной командой.

## Инструкция для вас (как подготовить архив):

1. Убедитесь, что у вас установлен Docker и Docker Compose.
2. Создайте архив проекта, **исключая** папки зависимостей (они будут загружены внутри Docker):
   ```bash
   # Пример команды для Linux/Mac:
   zip -r project.zip . -x "**/node_modules/*" "**/venv/*" "**/.next/*" "**/__pycache__/*"
   ```
3. Отправьте файл `project.zip` другу.

## Инструкция для друга:

1. Скачай и установи [Docker Desktop](https://www.docker.com/products/docker-desktop/).
2. Распакуй архив с проектом.
3. Открой терминал в папке проекта и запусти:
   ```bash
   docker compose up -d --build
   ```
4. Дождись завершения сборки. Когда всё будет готово, проект будет доступен по адресу:
   *   **Frontend:** [http://localhost:3000](http://localhost:3000)
   *   **Backend API:** [http://localhost:8000](http://localhost:8000)
